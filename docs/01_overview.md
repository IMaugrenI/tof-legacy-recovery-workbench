# Overview

`tof-legacy-recovery-workbench` is a reduced public-safe baseline for legacy recovery.

It is meant to show a disciplined flow for:
- reading older mixed material
- preserving provenance
- separating evidence from interpretation
- extracting reusable parts
- mapping outputs into a small stable target-class set

The repository is intentionally reduced. It demonstrates the method and the pipeline without exposing a private recovery corpus.
