# Pipeline

## Stage logic

### 00_input_alt
Input material enters unchanged.

### 01_intake
Every file becomes a structured intake record.

### 02_evidence
The system derives neutral evidence from intake outputs.

### 03_hypotheses
Open-set hypotheses remain suggestions, not final truth.

### 04_extracts
Mixed signals are split into parallel extract surfaces.

### 05_mapping
Outputs map into stable target classes.

### 06_review
Ambiguous or mixed cases are made visible for review.

### 07_reports
Coverage, duplicates, and acceptance are reported.
